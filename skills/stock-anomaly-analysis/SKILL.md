# 个股异动分析

从**游资、主力、机构、监管层、散户**五大维度综合分析个股异动的**触发因素**、**多方博弈格局**和**未来走势**。

## 第一原则：数据真实性 (最重要!)

**禁止幻觉生成数据。没有真实数据，就不生成报告。**

### 硬性要求

1. **每条信息必须来自WebSearch搜索结果**
   - 不得编造任何数据、日期、金额、URL
   - 搜索结果中没有的信息，就不写入报告

2. **每条信息必须有可验证的真实URL**
   - URL必须是WebSearch返回的原始链接
   - 禁止使用占位符如 `#demo` `#需替换` 等
   - 禁止编造公告ID、文章ID

3. **数据不足时的处理**
   - 若搜索不到近期触发因素：明确告知用户"未找到近期异动消息"
   - 若搜索不到资金数据：该模块显示"暂无数据"
   - 不要用虚假数据填充空白

### 数据验证流程 (必须执行!)

```
1. WebSearch 获取信息 → 记录返回的URL和摘要
2. WebFetch 验证关键URL → 确认内容与摘要匹配 (必须!)
3. 只有验证通过的数据才能写入报告
```

### WebFetch 验证规则

**必须验证的URL类型:**
- 龙虎榜数据URL (涉及具体金额)
- 公司公告URL (涉及具体事件)
- 重大订单/合同URL (涉及具体金额)
- 今日行情/异动URL (涉及当日数据)

**⚠️ URL必须是用户可浏览的网页! 禁止使用以下类型的URL:**
- API数据接口 (如 `qt.gtimg.cn/q=sz002195`、`push2.eastmoney.com/api/...`)
- 返回JSON/纯文本的接口
- 需要特殊客户端才能访问的URL

**正确的URL替换规则:**
```
API接口URL → 对应的可浏览网页URL

qt.gtimg.cn/q=sz002195 → https://stockpage.10jqka.com.cn/002195/
push2.eastmoney.com/api/... → https://data.eastmoney.com/zjlx/002195.html
datacenter-web.eastmoney.com/api/... → https://data.eastmoney.com/stock/lhb/002195.html
push2his.eastmoney.com/api/... → https://quote.eastmoney.com/sz002195.html
```

**验证步骤 (5步缺一不可!):**
```
1. WebFetch(url) 获取页面内容
2. ⚠️ 首先检查文章发布日期 (最重要!)
   - 寻找页面中的发布时间戳
   - 确认年份与分析日期一致
   - 例: 分析2026年2月5日 → 文章必须是2026年发布
3. 检查页面中是否包含:
   - 搜索摘要中提到的关键数据 (金额、日期、公司名)
   - 与标题匹配的核心内容
4. 若日期+内容都匹配 → 数据有效，记录为 verified: true
5. 若日期不匹配或内容不匹配 → 丢弃该数据，不写入报告
```

**⚠️ 年份验证是第一优先级!**

搜索"2月5日龙虎榜"可能返回去年同日的文章，必须核对发布年份！

**验证示例 (正确):**
```
分析日期: 2026年2月5日
WebSearch返回: "岩山科技龙虎榜，机构净买入7059万"
URL: https://finance.ifeng.com/c/xxx

WebFetch验证:
1. 页面发布时间: "2026年02月05日 16:41" ✓ (年份匹配!)
2. 页面包含"岩山科技" ✓
3. 页面包含"7059" ✓
4. 页面包含"机构" ✓
→ 验证通过，数据可用
```

**验证示例 (失败 - 年份错误):**
```

WebFetch验证:
1. 页面发布时间: "2025年02月05日 16:41" ✗ (年份不匹配!)

```

**验证示例 (失败 - 404/内容不匹配):**
```
若WebFetch显示"404"或内容不匹配:
→ 验证失败，丢弃该条数据
```

## 数据结构

**异动分析 = 近期触发 + 历史背景 + 多方博弈 + 技术形态**

- **triggers (近期触发)**: 最近1-3天的新消息，必须有真实URL
- **background (历史背景)**: 近几个月的重要事件，必须有真实URL
- **participants (多方博弈)**: 游资/主力/机构/监管/散户五维度分析
- **technical_pattern (技术形态)**: K线形态识别与量价关系


#### 2.2 市场广度与情绪温度

**搜索词:**
```
1. "A股 涨停 跌停 家数 [完整年月日]"
2. "今日涨停板 复盘 [完整年月日]"
3. "市场情绪 涨跌比 [完整年月日]"
```

**核心指标:**

| 指标 | 说明 | 判断标准 |
|------|------|---------|
| **涨停家数** | 当日涨停板数量 | >80只=极热, 40-80=活跃, 20-40=一般, <20=冷清 |
| **跌停家数** | 当日跌停板数量 | >30只=恐慌, 10-30=偏弱, <10=正常 |
| **涨跌比** | 上涨家数/下跌家数 | >3:1=强势, 2:1=偏强, 1:1=震荡, 1:2=偏弱 |
| **涨停封板率** | 封板涨停/曾涨停 | >70%=封板强, 50-70%=一般, <50%=烂板多 |
| **炸板率** | 炸板数/曾涨停 | <20%=强势, 20-40%=正常, >40%=弱势 |
| **两市成交额** | 沪深合计 | >1.5万亿=放量, 1-1.5万亿=正常, <8000亿=缩量 |

**情绪温度计:**
```
🔥 过热 (>80分): 涨停>80, 跌停<5, 涨跌比>4:1 → 警惕高位风险
😊 活跃 (60-80): 涨停40-80, 涨跌比>2:1 → 赚钱效应好
😐 一般 (40-60): 涨停20-40, 涨跌比≈1:1 → 结构性行情
😟 低迷 (20-40): 涨停<20, 跌停>涨停 → 亏钱效应蔓延
🥶 冰点 (<20):  涨停<10, 跌停>30 → 恐慌杀跌（可能见底）
```

#### 2.3 连板梯队分析

**搜索词:**
```
1. "今日连板 梯队 [完整年月日]"
2. "涨停板 连板股 高度板 [完整年月日]"
3. "财联社 涨停复盘 [完整年月日]"
```

**梯队结构:**
```
最高板 (龙头高度): 决定市场情绪上限
  ↓ 7板以上: 极强赚钱效应，市场疯狂
  ↓ 5-6板: 强赚钱效应，题材持续性好
  ↓ 3-4板: 正常赚钱效应
  ↓ 1-2板: 弱赚钱效应，短线难做

各级梯队:
  最高板 → 代表市场高度，龙头所在
  次高板 → 补涨或分支龙头
  3连板  → 中位接力区
  2连板  → 晋级淘汰区（关键观察位）
  首板    → 新题材试错区
```

**对个股的影响判断:**
```
个股在梯队中的位置:
  - 该股是几板? 在梯队中属于什么位置?
  - 同梯队竞争者有哪些? 谁更有辨识度?
  - 上方有没有空间? (最高板还能往上走吗?)
  - 梯队是否完整? (断层=接力风险大)

示例:
  最高板: 5板(AI概念A)
  4板: 2只(AI概念B, 机器人C)
  3板: 该股 + 3只竞品 ← 竞争激烈，需要辨识度
  2板: 8只
  首板: 25只 ← 新题材不断，活跃度高
```

#### 2.4 板块与风格分析

**搜索词:**
```
1. "[所属板块] 板块 涨跌幅 [完整年月日]"
2. "今日板块 涨幅排行 [完整年月日]"
3. "板块资金流向 [完整年月日]"
```

**板块分析要点:**
- **所属板块排名**: 该股所属板块在今日板块涨幅中的排名
- **板块领涨/领跌**: 板块内涨停家数，该股是否为板块龙头
- **板块联动**: 关联板块是否同步走强（如AI+机器人+芯片联动）
- **板块轮动**: 热点是否在轮动中? 该板块是刚启动还是已高位?
- **板块资金**: 板块整体资金流入还是流出

**市场风格判断:**
```
大盘 vs 小盘:
  上证50↑ + 中证1000↓ → 大盘风格 (机构主导)
  上证50↓ + 中证1000↑ → 小盘风格 (游资主导)
  全面↑                → 普涨行情
  全面↓                → 普跌行情

价值 vs 成长:
  银行/保险/白酒领涨 → 价值风格
  AI/新能源/半导体领涨 → 成长风格

对个股影响:
  该股属于什么风格? 当前市场风格是否有利?
  若市场风格不利，个股逆势上涨的持续性存疑
```

### 第五步: 多维度市场参与者分析 (核心!)

**从五大维度拆解异动背后的力量博弈，每个维度独立搜索、独立验证。**

#### 5.1 游资分析 (短线热钱)

**搜索词:**
```
1. "[股票名称] 游资 [完整年月日]"
2. "[股票名称] 涨停 龙虎榜 游资 [完整年月日]"
3. "[股票名称] 复盘 打板 [完整年月日]"
4. "site:taoguba.com.cn [股票名称] [完整年月]"       ← 淘股吧游资复盘帖
5. "淘股吧 [股票名称] 打板 接力 [完整年月日]"        ← 知名游资ID观点
6. "财联社 涨停复盘 [完整年月日]"                    ← 涨停板复盘快讯
```

**分析要点:**
- **龙虎榜席位识别**: 知名游资营业部（如拉萨系、赵老哥关联席位、佛山系等）
- **操作手法判断**: 打板（一字板/T字板/烂板接力）、低吸、半路板
- **买卖方向**: 游资是买方主力还是卖方出货
- **连板接力意愿**: 是否有知名游资接力意愿（首板/二板/三板不同策略）
- **席位历史战绩**: 该席位近期操作风格和成功率

**态度判断 (stance):**
- `强烈看多`: 知名游资大举买入，有连板意图
- `看多`: 游资介入但金额一般
- `中性`: 游资买卖参半
- `看空`: 游资大幅卖出/获利了结
- `未参与`: 无游资身影

#### 5.2 主力分析 (控盘大资金)

**搜索词:**
```
1. "[股票名称] 主力资金 [完整年月日]"
2. "[股票名称] 资金流向 大单 [完整年月日]"
3. "[股票名称] 控盘度 筹码分布"
```

**优先使用API数据:**
```bash

```

**分析要点:**
- **资金流向**: 主力净流入/流出金额及趋势（连续几日流入?）
- **大单/超大单**: 超大单（>100万）和大单（20-100万）的净买入情况
- **控盘度**: 主力控盘比例，筹码集中度
- **量价配合**: 放量上涨（主力进场）vs 缩量上涨（锁仓拉升）vs 放量下跌（主力出货）
- **成本分析**: 主力建仓成本区间，当前价位相对于成本的位置

**态度判断 (stance):**
- `强烈看多`: 主力连续大额流入，控盘度提升
- `看多`: 主力净流入，量价配合良好
- `中性`: 资金流向波动，方向不明
- `看空`: 主力持续流出，大单砸盘
- `强烈看空`: 主力大幅出货，筹码松动

#### 5.3 机构分析 (基金/保险/QFII等)

**搜索词:**
```
1. "[股票名称] 机构 持仓 [年份]"
2. "[股票名称] 基金 重仓 [最近季度]"
3. "[股票名称] 研报 评级 [完整年月]"
4. "[股票名称] 机构调研 [完整年月]"
5. "[股票名称] 北向资金 [完整年月日]"
6. "site:xueqiu.com [股票名称] 机构 分析"            ← 雪球机构投资者观点
7. "券商中国 [股票名称] 研报 [完整年月]"              ← 券商研报解读
```

**分析要点:**
- **机构持仓变动**: 最近季报中基金/社保/QFII/保险的持仓变化
- **龙虎榜机构席位**: "机构专用"席位的买卖方向和金额
- **研报与评级**: 近期券商研报的目标价和评级（买入/增持/中性/减持）
- **机构调研**: 近期是否有机构扎堆调研，调研关注点
- **北向资金**: 沪/深股通持股变化，外资态度

**态度判断 (stance):**
- `强烈看多`: 机构扎堆调研 + 多家券商上调评级 + 北向加仓
- `看多`: 机构龙虎榜净买入，或研报维持买入评级
- `中性`: 机构持仓稳定，无明显增减
- `看空`: 机构减持，或研报下调评级
- `强烈看空`: 机构集体减持 + 评级下调 + 北向撤退

#### 5.4 监管层分析 (政策/合规/行业导向)

**搜索词:**
```
1. "[所属行业] 政策 [完整年月]"
2. "[股票名称] 问询函 监管 [年份]"
3. "[所属行业] 国家标准 规划 [年份]"
4. "[股票名称] 证监会 交易所 [年份]"
5. "[所属行业] 补贴 扶持 支持 [完整年月]"
6. "site:stcn.com [所属行业] 政策 [完整年月]"        ← 证券时报政策报道
7. "site:cs.com.cn [所属行业] 规划 [年份]"            ← 中证报权威解读
8. "财联社 [所属行业] 政策 [完整年月日]"              ← 政策快讯第一时间
```

**分析要点:**
- **行业政策**: 国家/地方产业政策、补贴、扶持计划（利好）
- **监管动态**: 证监会/交易所问询函、关注函、立案调查（利空）
- **行业规范**: 新出台的行业标准、准入门槛变化
- **反垄断/合规**: 反垄断调查、数据安全审查等
- **IPO/再融资/并购审批**: 相关审批进展

**态度判断 (stance):**
- `强利好`: 重大产业政策支持，如补贴、纳入国家战略
- `利好`: 行业政策温和支持，审批顺利
- `中性`: 无明显政策变化
- `利空`: 收到问询函/关注函，行业收紧
- `强利空`: 立案调查、处罚、行业整顿

#### 5.5 散户分析 (市场情绪/群体行为)

**搜索词:**
```
1. "[股票名称] 股吧 [完整年月日]"
2. "site:xueqiu.com [股票名称] [完整年月日]"          ← 雪球讨论（偏理性散户）
3. "site:taoguba.com.cn [股票名称] 散户"              ← 淘股吧（偏短线散户）
4. "[股票名称] 散户 情绪"
5. "A股 市场情绪 [完整年月日]"
```

**多平台情绪对比:**
```
东方财富股吧: 最大散户聚集地，情绪最直接（主要参考）
雪球:         偏中长线投资者，分析质量较高
淘股吧:       偏短线客、打板族，情绪波动大
集思录:       偏量化/套利，理性程度最高
→ 综合多平台情绪可更准确判断市场温度
```

**分析要点:**
- **讨论热度**: 股吧/雪球发帖量变化，是否突然爆发
- **情绪倾向**: 看多 vs 看空比例，恐慌 vs 贪婪
- **散户资金流向**: 小单（<5万）净买入/卖出
- **跟风行为**: 是否存在追涨杀跌、打板情绪高涨
- **认知偏差**: 散户对利好/利空的理解是否存在偏差（可作为反向指标）

**态度判断 (stance):**
- `极度狂热`: 股吧一片看涨，跟风追高（警惕反向信号!）
- `乐观`: 散户情绪偏正面，适度跟进
- `分歧`: 多空争论激烈，观点分化
- `悲观`: 散户恐慌出逃，看空为主
- `极度恐慌`: 股吧哀嚎遍野（可能是抄底信号）

**⚠️ 散户情绪常作为反向指标:**
- 散户极度狂热时 → 往往接近阶段顶部
- 散户极度恐慌时 → 往往接近阶段底部

#### 5.6 综合博弈格局判断

**在五大维度分析完成后，综合判断:**

```
多方力量 = 游资(看多) + 主力(看多) + 机构(看多) + 监管(利好) + 散户(乐观)
空方力量 = 游资(看空) + 主力(看空) + 机构(看空) + 监管(利空) + 散户(悲观)
```

**博弈格局分类:**
- **多方碾压**: >=4个维度看多，强势上攻信号
- **多方占优**: 3个维度看多，偏强运行
- **多空均衡**: 各有2-3个看多/看空，震荡整理
- **空方占优**: 3个维度看空，偏弱运行
- **空方碾压**: >=4个维度看空，强势下跌信号

**特殊博弈场景:**
- 游资+散户看多 vs 机构+主力看空 → 短期可能冲高，但中期承压（聪明钱撤退）
- 机构+主力看多 vs 散户看空 → 逆势吸筹，关注建仓信号
- 监管利空 vs 其他维度看多 → 政策风险优先，谨慎对待

### 第六步: K线技术形态分析

**根据API获取的K线数据（第一步 `fetch_stock_data.py` 已获取），识别当前所处的技术形态。**

```bash
# 确保已获取近期K线
python scripts/fetch_stock_data.py [代码] --kline
```

**结合K线数据判断以下形态，将识别结果写入报告的 `technical_pattern` 字段。**

#### 看涨形态

**1. 多方炮（两阳夹一阴）**
```
形态: 阳线 → 阴线 → 阳线，三根K线组合
条件:
  - 第1根: 中/大阳线
  - 第2根: 小阴线，实体包含在第1根阳线实体内
  - 第3根: 中/大阳线，收盘价高于第1根阳线
  - 成交量: 两阳线放量，阴线缩量（量价配合）
含义: 多方进攻→空方试探→多方再攻，后续看涨
可靠度: ★★★★ (出现在上升趋势初期最可靠)
```

**2. 老鸭头**
```
形态: 股价缓慢上涨→小幅回落→再次上攻突破
条件:
  - 5日均线上穿10日均线（鸭颈）
  - 股价回落但不跌破10日均线（鸭头弧顶）
  - 5日均线回踩10日均线后再次金叉（鸭嘴）
  - MACD: 回落时不死叉或在零轴上方金叉
  - 成交量: 上涨放量→回落缩量→再涨放量
含义: 主力洗盘完毕，即将展开主升浪
可靠度: ★★★★★ (经典主力洗盘形态)
```

**3. 仙人指路**
```
形态: 长上影线的小阳/小阴线
条件:
  - 上影线长度 >= 实体的2倍
  - 出现在上升趋势中（重要前提!）
  - 当日成交量适中，非天量
  - 次日未大幅低开
含义: 主力向上试盘，探测上方压力，后续大概率上攻
可靠度: ★★★★ (必须在上升趋势中，下跌中则为"射击之星"看跌)
⚠️ 注意: 下跌趋势中的长上影线是"射击之星"，含义相反!
```

**4. 头肩底（反转形态）**
```
形态: 左肩→头部→右肩→突破颈线
条件:
  - 出现在下跌趋势末端
  - 三个波谷: 左肩、头（最低）、右肩（接近左肩高度）
  - 两个波峰连线构成"颈线"
  - 放量突破颈线确认反转
  - 右肩成交量通常小于左肩（空方力量衰竭）
含义: 下跌趋势结束，中期反转上涨信号
可靠度: ★★★★★ (经典底部反转形态)
目标位: 突破点 + (颈线位 - 头部最低价)
```

**5. 红三兵**
```
形态: 连续三根中小阳线
条件:
  - 三根阳线依次收高
  - 每根阳线开盘价在前一根实体内
  - 上影线较短（强势特征）
  - 出现在底部或整理末期
含义: 多方持续推进，后续看涨
可靠度: ★★★ (需结合量能判断)
```

**6. 早晨之星（启明星）**
```
形态: 大阴线 → 十字星/小实体 → 大阳线
条件:
  - 第1根: 大阴线（下跌末期）
  - 第2根: 跳空低开的十字星或小实体（犹豫）
  - 第3根: 大阳线，深入第1根阴线实体内
含义: 空方力量衰竭，多方开始反攻
可靠度: ★★★★ (底部反转信号)
```

**7. 底部双针探底**
```
形态: 两根带长下影线的K线
条件:
  - 间隔1-5根K线内出现两根长下影线
  - 两根下影线低点接近（支撑位确认）
  - 实体较小，下影线 >= 实体2倍
含义: 下方支撑位经过两次确认，短期底部成立
可靠度: ★★★★ (两次确认比单次更可靠)
```

#### 看跌形态

**8. 头肩顶（反转形态）**
```
形态: 左肩→头部→右肩→跌破颈线
条件:
  - 出现在上涨趋势末端
  - 三个波峰: 左肩、头（最高）、右肩（低于头部）
  - 两个波谷连线构成"颈线"

  - 头部成交量通常最大
含义: 上涨趋势结束，中期反转下跌信号
可靠度: ★★★★★ (经典顶部反转形态)
目标位: 跌破点 - (头部最高价 - 颈线位)
```

**9. 黄昏之星**
```
形态: 大阳线 → 十字星/小实体 → 大阴线
条件:
  - 第1根: 大阳线（上涨末期）
  - 第2根: 跳空高开的十字星或小实体
  - 第3根: 大阴线，深入第1根阳线实体内
含义: 多方力量衰竭，空方开始反攻
可靠度: ★★★★ (顶部反转信号)
```

**10. 三只乌鸦**
```
形态: 连续三根中阴线
条件:
  - 三根阴线依次收低
  - 出现在上涨后的高位
  - 开盘价在前一根实体内
含义: 空方持续打压，后续看跌
可靠度: ★★★ (高位出现更有意义)
```

**11. 乌云盖顶**
```
形态: 大阳线 → 大阴线
条件:
  - 阴线高开，深入阳线实体1/2以上
  - 出现在上涨趋势中
含义: 上方抛压沉重，短期见顶
可靠度: ★★★★ (阴线深入越深越可靠)
```

#### 中继/整理形态

**12. 上升三法**
```
形态: 大阳线 → 2-3根小阴线 → 大阳线
条件:
  - 小阴线群包含在首根阳线实体范围内
  - 末根阳线收盘创新高
  - 小阴线缩量，大阳线放量
含义: 上升趋势中的正常回调整理，后续继续上涨
可靠度: ★★★★
```

**13. 收敛三角形**
```
形态: 高点降低 + 低点升高，形成三角形
条件:
  - 至少2个依次降低的高点
  - 至少2个依次升高的低点
  - 成交量逐渐萎缩
  - 在三角形2/3处突破最有效
含义: 多空僵持，即将选择方向（向上突破看涨，向下突破看跌）
可靠度: ★★★ (需等待突破方向确认)
```

**14. 箱体震荡**
```
形态: 股价在明确的上下边界内反复波动
条件:
  - 至少2次触及上边界回落
  - 至少2次触及下边界反弹
  - 成交量在边界附近放大
含义: 主力震荡洗盘或建仓阶段
可靠度: ★★★ (突破方向是关键)
```

#### 形态分析注意事项

- **趋势是前提**: 同一形态在不同趋势中含义完全不同（如仙人指路 vs 射击之星）
- **量价配合**: 形态可靠性需要成交量确认，无量的突破不可信
- **多周期验证**: 日线形态需在周线级别得到确认
- **不要孤立看形态**: 必须结合五维度博弈分析综合判断
- **形态失败预案**: 每个形态都有失败条件，必须设置止损位

### 第七步: 预测走势

基于 **大盘环境 + 多维度博弈 + 技术形态** 综合判断:
- 大盘配合度: 市场环境是否支撑个股走势
- 短期 (1-5日): 支撑位、压力位，结合游资/主力动向 + K线形态 + 连板梯队空间
- 中期 (1-3月): 趋势判断，结合机构/政策面 + 周线形态 + 板块轮动周期
- 核心逻辑: 最看重哪个维度的信号（简述理由）
- 技术形态: 当前处于什么形态，后续大概率走向
- 风险提示: 列出各维度的潜在风险 + 形态失败条件 + 大盘风险

## 注意事项

- **⚠️ 年份验证是第一优先级**: WebFetch验证时，必须首先核对文章发布年份。搜索"2月5日"可能返回去年同日数据，必须核验年份一致!
- **WebFetch验证必须执行**: 关键数据URL必须经过WebFetch验证，确认日期+内容都匹配
- **搜索词必须包含完整年份**: 使用"2026年2月5日"而非"2月5日"
- **时效性优先**: 异动分析必须以最新消息为主
- **区分触发与背景**: 旧消息是背景，不是触发因素
- **来源可追溯**: 每条信息必须有日期和URL，且经过验证
- **验证失败即丢弃**: 若年份不匹配、404、或内容不匹配，该数据不得写入报告
- **数据不足坦诚告知**: 若找不到当日数据，明确告知用户，不要用旧数据冒充
- **免责声明**: 所有分析仅供参考，不构成投资建议
