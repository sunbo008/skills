# 趋势交易员培养体系 — A股闭环训练计划

> **设计理念：** 本计划按"知识输入 → 刻意练习 → 测试达标 → 实战检验 → 复盘迭代"的闭环设计，每个阶段设有明确的学习范围、指定资料、练习任务和通关标准。未达标不得进入下一阶段。
>
> **市场定位：** 本计划专为 **中国A股市场** 设计，充分考虑T+1交易制度、涨跌停板机制、主力控盘特征、政策驱动属性等A股独有的市场结构。

---

## 总览：培养路径图

```
阶段零          阶段一          阶段二          阶段三          阶段四          阶段五          阶段六
认知重塑  ──→  趋势识别  ──→  指标与盘口  ──→  规则制定  ──→  选股体系  ──→  模拟实战  ──→  持续进阶
(1周)          (2周)          (2周)           (2周)          (2周)          (4周)          (长期)
   │              │              │               │              │              │              │
   ▼              ▼              ▼               ▼              ▼              ▼              ▼
 笔试测验      视觉测试      指标+盘口测试    回测报告      选股评审      实盘考核      季度审计
 ≥80分         ≥90%准确     ≥85%准确        胜率≥40%      命中率≥30%   规则执行≥80%  持续盈利
```

**核心原则：** 趋势交易的精髓不是"预测"趋势，而是"跟随"趋势——等趋势确立后上车，趋势破坏后下车，中间的波动全部忽略。

### 缠论108课分阶段学习指引

> **缠中说禅（缠师）** 的《教你炒股票》108课是A股本土原创的最完整技术分析理论体系。缠论的核心框架是：**分型 → 笔 → 线段 → 中枢 → 走势类型 → 背驰 → 三类买卖点**，由底层几何结构递归构建出完整的市场描述语言。
>
> **学习原则：**
> - 缠论体系庞大，**必须配合本计划分阶段学习**，禁止一次性通读108课（会消化不良）
> - 每个阶段只学对应的课程，学完后必须结合该阶段的练习任务进行实操验证
> - 原始博文是唯一权威来源，市面上的"缠论解读"书籍质量参差不齐，慎选
> - 上班族阶段性目标：先掌握日线/周线级别的分型、笔、中枢、背驰和买卖点即可

### 缠论学习常见误区（必读警示）

| 误区 | 正确认知 |
|------|---------|
| 一上来就学全部108课 | **必须分阶段学**，108课是两年博文积累，信息密度极大，强行通读会一知半解 |
| 纠结于笔和线段的精确划分 | 初学阶段允许10-20%的划分误差，重要的是理解结构含义而非像做数学题一样精确 |
| 把缠论当万能预测工具 | 缠论的核心是"分类应对"而非"预测涨跌"，与趋势交易"跟随"理念一致 |

| 在分钟级别上反复操作 | 上班族在T+1市场中，分钟级别分析实用性极低。日线/周线级别的缠论分析已足够 |
| 过度追求"精确打击" | 缠论的区间套可以做到很精确，但精确需要大量实战经验积累，初学阶段用"大致区间"足矣 |

### 明确的学习边界（不学清单）

> 控制学习范围和当前计划同等重要。以下内容在整个培养期内**禁止涉猎**，避免分散注意力：

- ❌ 波浪理论（主观性太强，数浪千人千面；注意：缠论虽涉及走势分解，但其基于集合论的分型/笔/线段体系比波浪理论严谨得多，不在此禁止之列）
- ❌ 分钟级别K线（上班族无法盯盘，且A股T+1下分钟级操作无意义；缠论的多级别分析暂只学日线/周线级别，分钟级别留待阶段六进阶解锁）
- ❌ 短线打板/涨停板接力战法（这是游资的游戏，散户不具备资金和信息优势）
- ❌ 期权、期货、融资融券（在股票趋势交易成熟前不碰杠杆工具）
- ❌ 超过5个以上的技术指标（贪多嚼不烂）
- ❌ 任何"预测性"分析方法（如江恩理论、占星交易等）
- ❌ 深度财报分析/估值建模（趋势交易以技术面为主，但需做基本的**排雷筛查**——见下方说明）

> **关于"基本面排雷"的特别说明：** A股不同于成熟市场，存在大量财务造假、壳公司、僵尸企业。趋势交易虽不做基本面选股，但**必须做基本面排雷**——这不是"学基本面"，而是"不踩地雷"。排雷规则见阶段四选股体系。

## A股市场结构必修课（贯穿全程）

> 以下是A股区别于其他市场的核心机制，不理解这些机制就无法制定有效的交易规则。本内容在阶段零和阶段一中完成学习。

### T+1 交易制度

| 规则 | 对策略的影响 |
|------|-------------|
| 当日买入的股票，最早T+1（下一个交易日）才能卖出 | 买入决策必须更加慎重，无法当日纠错；如果买入后当日即大跌，只能隔日止损 |
| 当日卖出的资金可以当日再买入（资金T+0） | 止损后资金可以立即使用，但必须遵守冷静期规则 |

### 涨跌停板制度

| 板块 | 涨跌幅限制 | 策略影响 |
|------|-----------|---------|
| 主板（沪深） | ±10% | 趋势交易的主战场，流动性好 |
| 创业板（300开头） | ±20% | 波动更大，止损空间需加大 |
| 科创板（688开头） | ±20%（上市前5日无涨跌停） | 波动极大，新手阶段建议回避 |
| ST / *ST 股票 | ±5% | **绝对禁止交易**——流动性差，退市风险高 |
| 北交所 | ±30% | 流动性不足，新手阶段建议回避 |

**涨跌停对策略的关键影响：**
- 涨停封板时可能买不到（挂单排队），不要追涨停
- 跌停时可能卖不掉（无人接盘），止损可能无法执行——这是A股最大的风险之一
- 因此仓位管理和标的选择（流动性）比在其他市场更加重要

### 主力行为模型（A股核心认知）

> A股散户占比约 80% 的交易量，但持有市值不到 30%。主力（公募/私募/游资/外资）是真正的定价者。理解主力行为是A股生存的基础。

| 阶段 | 主力行为 | K线特征 | 散户常见错误 |
|------|---------|---------|-------------|
| **吸筹期** | 低位反复震荡，小幅收集筹码 | 底部放量但股价不涨，长期横盘 | 觉得"不涨"而卖出 |
| **洗盘期** | 制造恐慌，清洗浮筹 | 急跌破均线（甚至破MA60），但快速收回；缩量阴跌 | **被洗出去**（以为趋势结束） |
| **拉升期** | 快速拉升，吸引跟风盘 | 放量大阳线，均线多头排列 | 这是我们要参与的阶段 |
| **出货期** | 高位大量卖出，将筹码转移给散户 | 高位放巨量但涨幅收窄，长上影线频现 | **高位接盘**（以为还会涨） |

**关键认知：** 趋势交易在A股的核心挑战不是"找趋势"，而是**区分"主力洗盘"和"趋势结束"**。这决定了你是被洗出局还是成功持仓。

### 游资 vs 机构 vs 外资：不同玩家的特征

| 玩家 | 操作风格 | 持仓周期 | 对趋势交易者的影响 |
|------|---------|---------|-------------------|
| **游资** | 快进快出，做强势股/题材股，喜欢涨停板 | 1-5天 | 游资炒作的股票不适合趋势交易（持续性差），回避 |
| **公募基金** | 重仓蓝筹/成长股，抱团取暖 | 季度-年度 | 公募重仓股趋势持续性好，适合跟踪 |
| **私募基金** | 风格多样，部分有控盘能力 | 周-月 | 关注其持仓变化 |
| **北向资金（外资）** | 偏好高ROE蓝筹，对政策敏感 | 月-年 | 北向资金持续流入的个股趋势可靠性高 |
| **国家队（汇金/社保）** | 大盘极端下跌时托底 | 长期 | 国家队入场 = 底部信号（但不精确） |

### A股换手率判断标准（替代绝对成交量）

> 在A股，**换手率比绝对成交量更有意义**，因为不同股票流通盘大小差异极大。同样放量100亿，对茅台和对小盘股含义完全不同。

| 换手率 | 含义 | 趋势交易中的判断 |
|--------|------|-----------------|
| < 1% | 极度缩量，无人交易 | 观望或主力高度控盘 |
| 1-3% | 正常缩量 | 回踩阶段的健康缩量信号 |
| 3-7% | 温和放量 | 突破时的健康放量（最佳） |
| 7-15% | 明显放量 | 需要警惕：可能是启动信号，也可能是主力对倒出货 |
| > 15% | 巨量换手 | **高度警惕**——高位巨量换手大概率是主力出货（特别是涨幅已大的个股） |

### 洗盘 vs 出货识别（A股核心技能）

| 特征 | 主力洗盘 | 主力出货 |
|------|---------|---------|
| 所处位置 | 上涨初期或中期 | 大幅上涨后的高位 |
| 下跌速度 | 快速急跌（制造恐慌），1-3天完成 | 缓慢阴跌或高位反复震荡 |
| 成交量 | 缩量下跌（卖盘其实不大） | 放量下跌或高位持续放量 |
| 均线状态 | MA60 未走平或仍向上 | MA20 开始走平甚至拐头向下 |
| 关键K线 | 长下影线（下方有承接） | 长上影线（上方有抛压） |
| 回收速度 | 1-5日内快速收回失地 | 反弹无力，无法收回前高 |

### 交易规则定义（A股适配版）

#### 买入规则（满足全部条件才买）

| 编号 | 规则 | 判定标准 | A股适配说明 |
|------|------|---------|------------|
| B1 | 周线趋势向上 | 周K线在 MA20 上方，周 MACD 在零轴上方或金叉 | 同时要求大盘（上证/沪深300）周线不在MA20下方——A股系统性风险下个股难独善其身 |
| B2 | 日线出现回踩买点 | 股价回踩 MA20 获得支撑 | 允许盘中刺破MA60（主力洗盘常见），但**收盘价必须站回MA60上方** |
| B3 | 回踩缩量 + 企稳温和放量 | 回踩时换手率 < 3%；企稳时换手率 3-7% | 用换手率替代绝对成交量判断 |
| B4 | MACD 不死叉或即将金叉 | 日线 MACD 在零轴上方运行 | |
| B5 | 不在涨停板追买 | 已涨停的股票不挂单排队买入 | A股涨停追买风险极大——T+1无法当日止损，次日若低开则直接亏损 |

#### 卖出规则（满足任一条件就卖）

| 编号 | 规则 | 判定标准 | A股适配说明 |
|------|------|---------|------------|
| S1 | 跌破 MA20 且 3 日收盘不收回 | 连续3个交易日**收盘价**低于MA20 | 强调"收盘价"——盘中刺破不算，因为主力洗盘经常盘中打压尾盘拉回 |
| S2 | 周线 MACD 死叉 | 周线DIF下穿DEA | |
| S3 | 趋势走坏的综合信号 | MA20走平或拐头向下 + 换手率持续放大 + 反弹无力 | 替代原来的固定百分比止盈——趋势交易不应限制上涨空间 |
| S4 | 高位出货信号 | 大涨后出现：换手率 > 15% + 长上影线 + 股价不再创新高 | 连续2-3天出现此特征才确认，单日可能是洗盘 |

> **关于止盈的重要修正：** 原计划的"盈利20%出一半，30%全出"是典型的散户思维，会严重损害趋势交易的盈亏比。A股趋势行情中，龙头股涨幅经常超过50-100%。**趋势交易不设固定止盈点，用趋势走坏信号（S1-S4）作为出场依据。** 如果实在不放心，可以在盈利30%后将止损位上移到成本价（保本止损），但不主动止盈。

#### 止损规则（铁律）

| 编号 | 规则 | 判定标准 | A股适配说明 |
|------|------|---------|------------|
| L1 | **收盘价**跌破 MA60 且次日低开不收回 | 不以盘中价为准，而以收盘价为准 | A股主力洗盘最常用的手法就是盘中快速打到MA60以下，吓出散户后快速拉回。以收盘价判断可过滤大部分洗盘 |
| L2 | 单笔亏损不超过总资金的 **3%**（主板）/ **5%**（创业板） | 买入前必须计算止损位和仓位 | 创业板波动更大（±20%涨跌停），止损空间需更宽，因此仓位需更小 |
| L3 | 止损后冷静期 ≥ 3 个交易日 | 物理隔离：不看盘 | T+1制度下，止损最早在T+1执行。如果T+1跳空低开已超过止损位，按开盘价止损，不犹豫 |
| L4 | 跌停板止损策略 | 如果持仓股跌停且无法卖出，次日集合竞价阶段挂跌停价卖出 | 这是A股独有的风险——跌停时可能卖不掉，必须提前有预案 |

### 仓位管理规则（A股版）

| 规则 | 标准 | 说明 |
|------|------|------|
| 单只个股最大仓位 | ≤ 20% | A股个股波动大，集中持仓风险极高 |
| 同一板块最大仓位 | ≤ 30% | A股板块联动性极强，同板块等于同一笔交易 |
| 同时持股数量 | 3-5只 | 上班族精力有限，超过5只无法跟踪 |
| 创业板/科创板总仓位 | ≤ 30% | 波动大，需要控制整体风险敞口 |
| 大盘弱势（MA20下方）时总仓位 | ≤ 30% 或空仓 | A股系统性风险下，个股很难逆势上涨 |

| 5. 个股筛选 | 从重点板块中筛选：周线多头排列的个股 | 满足B1（周线趋势向上）| 同花顺/东方财富 | 15分钟 |
| 6. 排雷筛查 | 对初筛股票做基本面排雷检查 | 排雷清单全部通过 | F10 | 10分钟 |
| 7. 龙头确认 | 在同板块候选股中，优先选涨幅领先、换手率健康的龙头 | 板块内涨幅靠前，且非纯游资炒作 | 龙虎榜辅助判断 | 5分钟 |
| 8. 制定计划 | 每只候选股写买入计划 | 包含：计划买入价、止损价、仓位、所属板块 | Excel/飞书表格 | 10分钟 |

### 龙虎榜辅助判断（识别"该不该参与"）

| 龙虎榜特征 | 含义 | 对趋势交易者的建议 |
|-----------|------|-------------------|
| 买入席位以机构为主 | 机构看好，有中期逻辑支撑 | 可以关注，趋势持续性较好 |
| 买入席位以知名游资为主 | 短线炒作为主，持续性存疑 | 谨慎——等游资退场后如果趋势仍在才考虑 |
| 买卖双方均为游资 | 游资对倒，纯短线博弈 | **回避**——趋势交易者不参与这种票 |
| 卖出席位出现机构 | 机构在离场 | **警惕**——可能是趋势结束的前兆 |

## 常见心理陷阱 & 应对策略（A股版）

| 陷阱 | 本质 | 应对规则 |
|------|------|---------|
| 小反弹追高 | 害怕错过（FOMO） | 检查B1-B5是否全部满足，不满足就不买 |
| 大反转不敢买 | 恐惧主导决策 | 只看规则不看情绪；条件满足就机械执行 |
| 频繁交易 | 行动偏误（用交易缓解焦虑） | 每月交易上限8笔，超出需写书面理由 |
| 不肯止损 | 损失厌恶 | 止损是系统在正常运行，执行止损 = 你在赢 |
| 赚小钱就跑 | 确定性偏好 | 趋势交易靠大赚弥补小亏，按S1-S4出场，不设固定止盈 |
| 亏损后加仓摊薄 | 赌徒谬误 | 绝对禁止；已触发L1-L4就执行，不补仓 |
| **被洗盘吓出后不敢再买回** | A股特有的心理创伤 | 如果止损后股票重新满足B1-B5，可以重新买入——被洗出不丢人，不敢买回才是真正的损失 |
| **听消息/跟大V操作** | 放弃独立判断 | 任何外部信息都必须通过你的规则系统验证，不过规则的一律忽略 |
| **政策利好就冲进去** | 对"政策市"的过度反应 | 政策利好只决定"关注什么板块"，具体买卖仍由B1-B5/S1-S4规则决定 |

## 附录：本计划与美股体系的关键差异

> 如果你之前学过欧美交易体系，以下差异点需要特别注意，避免水土不服。

| 维度 | 美股/成熟市场 | A股 | 本计划的适配 |
|------|-------------|-----|-------------|
| 交易制度 | T+0，可当日买卖 | T+1，买入后最早次日卖出 | 强调慎重买入、尾盘操作、跌停预案 |
| 涨跌限制 | 无涨跌停（有熔断机制） | 主板±10%，创业板±20% | 增加涨跌停对策略的影响分析 |
| 做空机制 | 散户可方便做空 | 融券极难、成本高、券源少 | 不考虑做空策略，只做多 |
| 市场参与者 | 机构为主（~90%） | 散户交易量占~80% | 增加主力行为学、洗盘识别 |
| 市场驱动力 | 企业盈利驱动 | 政策+资金+情绪驱动 | 增加政策面分析、北向资金跟踪 |
| 信息披露 | 较为透明 | 信息不对称严重 | 增加基本面排雷机制 |
| 退市风险 | 退市常态化 | 退市逐步加速，但历史遗留问题多 | 增加ST/财务排雷一票否决 |
| 量能分析 | 用绝对成交量 | 用换手率更科学 | 全面用换手率替代绝对成交量 |
| 止损执行 | 可即时止损 | T+1 + 跌停限制可能导致无法止损 | 增加跌停预案（L4）和仓位控制 |
